# 📱 **CLTinder Corporativo — Match de Talentos**

Uma plataforma inspirada na dinâmica de um **app de relacionamento**, adaptada para o ambiente **corporativo**.

## 🎯 **Objetivo**

O objetivo do projeto é conectar profissionais e empresas de forma rápida, intuitiva e interativa, utilizando o conceito de match baseado em:

- ✅ **Interesses**
- ✅ **Habilidades** 
- ✅ **Oportunidades**

## 🚀 **Como Funciona**

Quando há interesse mútuo, ocorre o match, abrindo espaço para conexão e possíveis oportunidades profissionais.
